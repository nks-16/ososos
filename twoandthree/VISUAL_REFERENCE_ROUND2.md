## Key Formulas

### Need Calculation
```
Need[i][j] = Max[i][j] - Allocation[i][j]

For all processes i and resources j
```

### Available Calculation
```
Available[j] = Total[j] - Î£ Allocation[i][j]
                          i=0 to n-1

Where n = number of processes
```

### Safety Condition
```
For process Pi to execute:
Need[i][j] â‰¤ Work[j]  for all j

Where Work initially = Available
```

### Resource Release
```
When process Pi completes:
Work[j] = Work[j] + Allocation[i][j]  for all j
Allocation[i][j] = 0  for all j
```

## Quick Reference Commands

When using the application:

1. **Select Process**: Click on process node
2. **Request Resources**: Enter values, click "Request Resources"
3. **Check Safety**: Click "Check Safety" button
4. **View History**: Click "Show History"
5. **Reset**: Click "Reset Round" to start over

## Mathematical Notation Summary

```
n = number of processes (7)
m = number of resource types (4)

Allocation[n][m] = current allocation matrix
Max[n][m] = maximum demand matrix
Need[n][m] = remaining need matrix
Available[m] = available resource vector
Work[m] = working copy of available (used in safety algorithm)
Finish[n] = completion status array
```
